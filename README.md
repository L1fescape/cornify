# Cornify

Cornify spreads eternal joy and happiness across the web by adding rainbows and unicorns to any web page.

## Install
 `bower install cornify`

## Usage

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <title>Fabulous</title>
  <script src="bower_components/cornify/cornify.js"></script>
</head>
<body>
  <script>
    cornify.pizzazz();
  </script>
</body>
</html>
```

## Functions

##### Add some sparkle

`cornify.add()`

##### Take it away

`cornify.clear()`

##### Party time

`cornify.pizzazz()`
