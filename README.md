# Cornify

Cornify spreads eternal joy and happiness across the web by adding rainbows and unicorns to any web page.

## Install

```
npm install cornify-technology
```

## Functions

##### Add some sparkle

`cornify.add()`

##### Take it away

`cornify.clear()`

##### Party time

`cornify.pizzazz()`
